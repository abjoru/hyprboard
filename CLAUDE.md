# HyprBoard

A PureRef/BeeRef-style reference-image board, built in Rust with egui + eframe. See `CONTEXT.md` for the domain glossary.

## Agent skills

### Issue tracker

Issues and PRDs live as GitHub issues (`abjoru/hyprboard`); use the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Canonical triage roles map to this repo's label strings; the AFK-ready role is `afk` (the `afk.sh` runner's contract). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
